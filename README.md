# Robot Arm Control System (C++)

## 📌 Overview
This project implements a robot arm control system using C++ with a **state-based control structure (FSM)**.

The system controls multiple joints (L1, L2) and a gripper, executing operations in a **phase-based sequence**.

---

## ⚙️ Features
- Independent Joint Control (L1, L2)
- Gripper State Control (OPEN / CLOSE)
- Phase-based Operation (Step-by-step execution)
- Finite State Machine (FSM) design
- Modular C++ Class Structure
